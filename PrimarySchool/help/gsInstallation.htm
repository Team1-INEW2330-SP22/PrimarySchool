Installation

Please following these instructions to install the PrimarySchool application on your computer:

1.Download the latest installation package of the PrimarySchool Application and save to your computer

2.Extract the setup.exe file from the application's installation folder

3.Run the setup.exe file and follow the prompts during the installation wizard

4.Prior to running the application, ensure the computer is connected to the internet

5.Run the PrimarySchool application and login to start a session

 

Contact the PrimarySchool support at inew2330team1@gmail.com if there is difficulty during the initial login.
