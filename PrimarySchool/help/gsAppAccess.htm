Application Access

The PrimarySchool application is access protected and requires a user to input their credentials in order to gain access to the application. Only authorized credentials will allow the user to proceed past the login form.

Entering Credentials

In the form, enter the Username and password in the appropriate fields. By default, the user’s password is hidden as it is typed. Clicking the “eye” icon next to the password field will unhide the password. Click the “Log In” button to submit your credentials. If the submitted credentials are authorized, the Login Form will close and the Home Form will be displayed. Unauthorized submittals will result in an error message being displayed.

Resetting the User Password

To reset the user’s password, the application provides a “Reset Password” feature that can be accessed from the Login Page.

1.Select the "Forgot your password?" below the password field.

2.Enter the email addressed associated with the user's account.

3.Click the "Send Code" button.

If a matching email address is found in the user database, a email will be generated with a unique code and sent to the provided email address.

4.Enter the code within the email submitted.

5.Click the "Submit" button.

Upon success, the user will be prompted to enter a new password.

6.Enter the new password in the first field provided.

7.In the "Confirm Password", re-enter the new password.

8.Click the "Submit" button.

After successfully updating their password, the user will be returned to the Login Form and prompted to enter their credentials.
