Seating Charts

Seating Chart Design

The seating chart form is seperated into three sections including the course selection and details, student seat assignment list, and the detials of the room's available seats.

A course can been selected by using the Select Course dropdown field and selected the course. Selecting a course will display the course details including the course name, assigned teacher, the assign room, and the total number of available seats in the room. It will also populate the seat list.

There are two means to set seating assignments for the course:

• Manual Assignment: Available seats can be manually entered into the Seat ID field of the respective student registered for the course.

• Randomize Assignment: Use the Edit menu and select Randomize option or CTRL+SHIFT+R to have the seats randomly assigned to the registered students in the course.

Clearing / Reset Attendance

•Clear: Clears all seating assignments for the course selected.

•Reset: Reset all seating assignments for the course to the last saved settings.
