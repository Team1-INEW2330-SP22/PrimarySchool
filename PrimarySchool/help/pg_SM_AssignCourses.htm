Registering a Student

Assigning a Course

To assign a course to a student, use the navigation buttons or the search function to find the student’s record. For more information on how the use the search function, see the Searching for a Student section of this help file.

Once the student record is displayed, use the Available Courses list to find the course to assign to the student.

Select the course and click the “Add Course” button. The course will be moved to the This Student’s Courses list indicating the course has been assigned to the student.

Removing a Course Assignment

To remove an assigned course from the student, user the This Student’s Courses list to select the course to be removed.

Once selected, click the removed button. The course will be moved back to the Available Courses list indicating the course is now available to be assigned.
