Managing Assignments

The assignment form allows for the management of assignments and allows the assignment to be created, edited, added to an existing course, or removed for the application.

Create New Assignments

• Select the 'Create New' button on the form to create a new assignment. This will open a new record on the form to be populated.

• Complete the form including the Name, Category ID, and Description. The Category ID is provided in the list provided. This field is used to calculate the weighted average of the assignment class.

• When all fields are populated, save the record by selecting the Save button.

Updating Assignments

• To edit an existing assignment, use the navigation buttons or the search function to find the assignment record needing to be edited. Click the Edit button.

• When edits to all fields are completed, save the record by selecting the Save button.

Adding Assignments to Courses

• Click the Add to Course button to add the assignment to the course selected in the previous screen (Gradebook Form).

Removing Assignments

• Click the Remove from Course button to remove the assignment from the course selected in the previous screen (Gradebook Form).
