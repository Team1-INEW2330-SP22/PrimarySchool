Searching for a User

Within the User Form, a search field is available to find a specific user. The search function limits the displayed used by filter by the user’s last name.

In the ‘Search by Last Name’ field, enter the user’s last name and select the ‘Search’ button.

If more than one user has the last name entered in the search field, use the navigation button to cycle through the displayed records for the desired user profile.
