Editing as Existing User

To create a new user, navigate to the User Form from the Main Menu by going to Administrator -> Users.

 

Find the user by cycling through the profile records using the navigation buttons or by searching for the user using the search function. For more information on how the use the search function, see the Searching for a User section of this help file.

When the record is identified, select the ‘Edit’ button to release the hold on the record.

Make the necessary edits to the user’s profile. Click ‘Save’..
