Managing Teacher Course Assignments

Assigning a Course

To assign a course to a teacher, use the navigation buttons or the search function to find the teacher’s record. For more information on how the use the search function, see the Searching for a Teacher section of this help file.

Once the teacher record is displayed, use the Available Courses list to find the course to assign to the teacher.

Select the course and click the “Add Course” button. The course will be moved to the This Teacher’s Courses list indicating the course has been assigned to the teacher.

Removing a Course Assignment

To remove an assigned course from the teacher, user the This Teacher’s Courses list to select the course to be removed.

Once selected, click the removed button. The course will be moved back to the Available Courses list indicating the course is now available to be assigned.
