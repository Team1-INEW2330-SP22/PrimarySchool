Search Courses

Within the Course Form, a search field is available to find a specific course. The search function limits the displayed used by filter by the course’s name.

In the ‘Search by Course Name’ field, enter the course’s name and select the ‘Search’ button.

If more than one course has the name entered in the search field, use the navigation button to cycle through the displayed records for the desired course record.
