System Access

The PrimarySchool application is access protected and requires a user to input their credentials in order to gain access to the application. Only authorized credentials will allow the user to proceed past the login form.

Entering Credentials

In the form, enter the Username and password in the appropriate fields. By default, the user passwords are hidden as the characters are entered. Clicking the “eye” icon next to the password field will unhide the password. Click the “Log In” button to submit your credentials. If the submitted credentials are authorized, the Login Form will close and the Home Form will be displayed. Unauthorized submittals will result in an error message being displayed.

Exiting the Program

For security purposes, the application can only be exited via the main form that is view immediately after logging into the application. By clicking the X in the top right corner or by selecting File -> Exit in the main menu, the application will close and user is logged out of the application.
