Print Grades

Printing the Gradebook

The gradebook for each course can be exported to a PDF document with all available data for the course.

1. To print the gradebook, open the File menu and select the Print Gradebook option or press CTRL+P.

2. To print the midterm grades, open the File menu and select the Print Midterm option or press CTRL+M.
