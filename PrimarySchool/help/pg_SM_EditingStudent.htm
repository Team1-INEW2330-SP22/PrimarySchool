Editing an Existing Student

To edit an existing student, use the navigation buttons or the search function to find the student record needing to be edited. For more information on how the use the search function, see the Searching for a Student section of this help file.

 

When the record is identified, select the ‘Edit’ button to release the hold on the record.

Make the necessary edits to the student’s profile. Click ‘Save’.
