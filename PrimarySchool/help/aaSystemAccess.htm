System Access

The PrimarySchool application is access protected and requires a user to input their credentials in order to gain access to the application. Only authorized credentials will allow the user to proceed past the login form.

Entering Credentials

In the form, enter the Username and password in the appropriate fields. By default, the user’s password is hidden as it is typed. Clicking the “eye” icon next to the password field will unhide the password. Click the “Log In” button to submit your credentials. If the submitted credentials are authorized, the Login Form will close and the Home Form will be displayed. Unauthorized submittals will result in an error message being displayed.
