Update Attendance

Attendance Design

The attendance form is seperated into two sections including the course selection and details, and a data grid view of all students registered for the course with attendance details.

A course can been selected by using the Select Course dropdown field and selected the course. By default, the Select Date field is populated with today's date. However, if a different date needs to be entered or edited, the date can be changed by using the dropdown. The Attendance section will display the name of the course, the instructor's name, the room number, and the attendance date selected.

Recording Attendance

The attendance of each student can be recoded by selecting the Present, Excused, and Absence Reason fields.

•Present: Check the Checkbox to mark the student present. If the student is absent, leave this box unchecked.

•Excused: Check the Checkbox to mark the student's absence as excused.

•Absence Reason: Field to provide details regarding the student's absence.

When all updates have been completed, save the records by using File -> Save or CTRL+S..

Clearing / Reset Attendance

•Clear: Clears all attentance records for the course and date selected.

•Reset: Reset all attentance records for the course and date selected to the last saved settings.
