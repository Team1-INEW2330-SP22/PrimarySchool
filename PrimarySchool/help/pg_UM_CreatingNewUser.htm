Creating a New User

To create a new user, navigate to the User Form from the Main Menu by going to Administrator -> Users.

 

By default, the form is blank but is not ready to accept new information. Start by clicking the ‘Add New’ button.

Complete the form with the information of the user to add. Below are some requirements for each field.

•All fields are required

•Date of Birth: Must be in the date Format YYYY-MM-DD

•Zip: Must be 5 characters in length

The User ID field is an automatically generated number by the application and cannot be changed or selected..

Once all fields are entered, click the Save button to complete the entry..
