Create & Edit Courses

Creating a new Course

To create a new course, navigate to the Courses form from the main menu by going Academic Officer -> Courses.

 

Within the form, buttons are avialble for navigation, editing, saving, adding, and deleting courses.

1. To add a new course to the application, click the “Add New” button. This will enable the records fields and allow the entry of a new course record.

2. Complete the form with the information of the course to add. Below are some requirements for each field:

• All fields are required

• Name: Enter a unique name the course.

• Description: Enter a description of the course to provide details regarding subject manner and course structure.

• Teacher ID: Enter the numeric number associated to the teacher to assign to the course.

3. Students can be assigned to the course by using the available list of students in the 'Available Students' list. Select the student's name and click the 'Add Student' button to assign the student to the course.

4. To remove a student, select the student's name from the list of students in the 'Registered Students' list and click the 'Remove' button to remove the student to the course.

5. When complete, click the save button to save the record.

Editing a Course

To edit an existing course, navigate to the courses by cycling through the records or by using the search field.

1. Click the “Edit” button to unlock the record.

2. Modify the record by following the same criteria in steps 2 through 5 listed above.
