Editing as Existing Teacher

To edit an existing teacher, use the navigation buttons or the search function to find the teacher record needing to be edited. For more information on how the use the search function, see the Searching for a Teacher section of this help file.

 

When the record is identified, select the ‘Edit’ button to release the hold on the record.

Make the necessary edits to the teacher’s profile. Click ‘Save’.
